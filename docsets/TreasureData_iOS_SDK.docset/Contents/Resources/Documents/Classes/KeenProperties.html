KeenProperties

@interface KeenProperties : NSObject

This class is used to represent Keen-specific properties that can be included on each event. Today this includes timestamp and location.

  • timestamp

    The time a particular event occured. The Keen Client will automatically generate this for you, but you can set it yourself if you’d like.

    Declaration

    Objective-C

    @property (readwrite, strong, nonatomic) NSDate *timestamp;

    Swift

    var timestamp: Date! { get set }
  • location

    The location where a particular event occured. The Keen Client will automatically generate this for you, but you can set it yourself if you’d like.
